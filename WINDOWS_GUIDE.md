## 二、下载与安装

### 方式 A：从源码安装（推荐）

1. **下载项目源码**

   从 GitHub 下载最新版：
   > https://github.com/vincent9306/law-case-manager

   点击绿色「Code」按钮 → 「Download ZIP」，解压到任意目录（建议放在桌面或 D 盘根目录，不要放在包含中文或空格的路径中）。

2. **安装 Python**

   如果电脑尚未安装 Python，请访问以下地址下载并安装：
   > https://www.python.org/downloads/

   > ⚠️ **安装时务必勾选「Add Python to PATH」**（添加到系统路径），否则后续命令无法执行。

   安装完成后，验证 Python 是否可用：
   - 按 `Win + R`，输入 `cmd`，回车打开命令提示符。
   - 输入 `python --version`，如果显示版本号（如 `Python 3.12.0`），说明安装成功。

3. **使用自动安装脚本**

   - 在解压后的项目文件夹中，找到 `install_windows.ps1`。
   - 右键点击该文件，选择「**使用 PowerShell 运行**」。

   > 如果提示「无法加载，因为在此系统上禁止运行脚本」，请先以管理员身份打开 PowerShell，执行：
   > ```powershell
   > Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   > ```
   > 然后重新右键运行 `install_windows.ps1`。

   脚本会自动完成以下操作：
   - 检查 Python 环境
   - 安装所需依赖包（Flask、PyPDF2、openpyxl 等）
   - 创建数据目录和数据库
   - 生成 `start.bat` 启动文件

4. **启动系统**

   安装完成后，直接双击 `start.bat` 即可启动。浏览器将自动打开系统首页。

### 方式 B：手动安装

如果自动脚本无法运行，可以手动执行以下步骤：

1. 打开命令提示符（`Win + R` → `cmd`），进入项目目录：
   ```cmd
   cd /d D:\你的项目路径\law-case-manager
   ```

2. 安装依赖包：
   ```cmd
   pip install -r requirements.txt
   ```

3. 初始化数据库：
   ```cmd
   python -c "from app import app; from models import init_db; init_db()"
   ```

4. 启动程序：
   ```cmd
   python app.py
   ```

5. 浏览器访问 **http://127.0.0.1:5066**。

### 方式 C：打包为独立可执行文件

如果你希望生成一个不依赖 Python 环境的 `.exe` 文件：

1. 进入项目目录，双击「**Windows一键打包.bat**」。
2. 脚本会自动完成所有打包步骤。
3. 打包完成后，可执行文件位于 `dist\案件管理系统\` 目录下。
4. 将整个 `dist\案件管理系统\` 文件夹复制到任意位置使用。

> ⚠️ **重要提示**：如果运行 `.bat` 文件后出现乱码或报错（如 `'PDF2' 不是内部或外部命令`、`'cho' 不是内部或外部命令`），说明文件编码出了问题。此时请改用 **PowerShell 脚本**方式安装（方式 A 中的第 3 步），或尝试用记事本打开 `.bat` 文件，另存为时选择编码为「**ANSI**」。

## 三、首次启动

第一次启动时，系统会自动：

1. 在 `data\` 目录下创建 SQLite 数据库（`cases.db`）。
2. 在 `data\uploads\` 目录下创建文件上传目录。
3. 启动 Flask Web 服务器（默认端口 **5066**）。

启动成功后，命令提示符窗口会显示：

```
========================================
  个人案件管理系统
  Developed by Yizhen Li
  Beijing Dacheng Law Offices, LLP (Wuxi)
========================================

  正在启动服务...
```

浏览器将自动打开首页。若未自动打开，请手动访问 **http://127.0.0.1:5066**。

> ⚠️ **请勿关闭命令提示符窗口**——关闭窗口即停止服务。

## 五、常见问题排查

### Q1：双击 .bat 文件后出现乱码

**原因**：Windows 命令提示符默认使用 GBK 编码，如果 `.bat` 文件是 UTF-8 编码，就会显示乱码。

**解决方法**：
- **方法 1（推荐）**：使用 `install_windows.ps1`（PowerShell 脚本）代替 `.bat` 文件。右键选择「使用 PowerShell 运行」。
- **方法 2**：用记事本打开 `.bat` 文件，点击「文件 → 另存为」，在底部的「编码」下拉菜单中选择「ANSI」，保存替换原文件。
- **方法 3**：在命令提示符中手动输入 `chcp 65001` 回车后再运行 bat 文件。

### Q2：提示「python 不是内部或外部命令」

**原因**：Python 未安装，或未添加到系统 PATH 环境变量。

**解决方法**：
1. 重新运行 Python 安装程序。
2. 在安装界面的第一页，**务必勾选「Add Python to PATH」**。
3. 安装完成后，重新打开命令提示符，输入 `python --version` 验证。

如果已安装但未勾选 PATH：
1. 搜索「环境变量」→ 点击「编辑系统环境变量」。
2. 在「系统变量」中找到「Path」，点击「编辑」。
3. 添加 Python 安装路径（如 `C:\Users\你的用户名\AppData\Local\Programs\Python\Python312\`）和 `Scripts` 子目录。

### Q3：端口 5066 被占用

**解决方法**：
```cmd
:: 查找占用端口的进程
netstat -ano | findstr :5066

:: 记下最后一列的 PID，然后终止进程
taskkill /PID 进程号 /F
```

或修改 `app.py` 中最后一行的端口号（搜索 `5066`，替换为其他端口如 `5088`）。

### Q4：防火墙弹出拦截提示

首次启动时，Windows 防火墙可能会弹出网络访问提示。请点击「允许访问」，否则浏览器无法打开系统。

### Q5：浏览器打开后页面空白

1. 确认命令提示符窗口没有报错信息。
2. 确认地址栏输入的是 `http://127.0.0.1:5066`（注意是 **http** 不是 https）。
3. 尝试使用 Chrome 浏览器。
4. 关闭代理 / VPN 后重试。

### Q6：如何卸载

直接删除项目文件夹即可。系统不会在注册表或其他位置留下任何痕迹。如果需要保留数据，请先备份 `data\` 文件夹。

## 六、数据备份与恢复

数据库文件和上传文件位于 `data\` 目录下：

```
data\
├── cases.db          ← 数据库文件（所有案件数据）
└── uploads\          ← 上传的文件
```

**备份**：直接复制整个 `data\` 文件夹到安全位置（如 U 盘、云盘）。

**恢复**：将备份的 `data\` 文件夹覆盖回项目目录即可。

---

## 七、数据导出

系统支持将案件列表导出为 Excel 文件：

1. 进入「案件列表」页面。
2. 点击「导出 Excel」按钮。
3. 浏览器会自动下载 `.xlsx` 文件。

---

## 八、停止与重启

- **停止**：在命令提示符窗口按 `Ctrl + C`，或直接关闭窗口。
- **重启**：重新双击 `start.bat`（或你创建的启动文件）。

---

## 九、创建桌面快捷方式

1. 在项目文件夹中找到 `start.bat`。
2. 右键点击 →「发送到」→「桌面快捷方式」。
3. 双击桌面快捷方式即可快速启动。
